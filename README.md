# QordeR

QorderR is a restaurant order tracker that makes it easy for customers to order through a unique QR code.

## Dependencies

- axios
- bcrypt
- body-parser
- bootstrap
- cookie-session
- express
- hookrouter
- qrcode
- react
- react-cookie
- react-dom
- react-router-dom
- react-scripts
- react-stripe-checkout
- react-stripe-elements
- reactstrap
- stripe
- typeface-roboto
- @material-ui/core
- @material-ui/icons
- material-table
- ws


### Prerequisites

Must have Node JS installed into your host machine

[Node JS](https://nodejs.org/en/download/)

## Getting Started

Clone this repositary into your local machine


### Installing

once you clone the repository, open the directory and install all the dependencies with "npm install"

```
npm install
```
once all the dependencies installed
"npm start" to run application on the localhost:3000

```
npm start
```
